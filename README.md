🛠️ Tech Stack & Algorithms

Language: Python

Libraries: Pandas, NumPy, Scikit-learn, LightGBM, Streamlit

Machine Learning Models Used:

LightGBM Classifier (Deployed for Web App)

Random Forest Classifier

📋 Project Highlights

Handled missing weather data using advanced imputation (Median/Mode).

Performed Data Encoding for categorical variables.

Scaled numerical features using StandardScaler.

Achieved high accuracy using LightGBM.

Interactive Web UI for real-time predictions.

💻 How to Run Locally

Clone this repository.

Install the required libraries:

pip install -r requirements.txt


Run the Streamlit app:

streamlit run app.py
